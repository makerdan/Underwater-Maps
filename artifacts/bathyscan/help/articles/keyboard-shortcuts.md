# Keyboard Shortcuts

Every key BathyScan responds to, in one place.

## Camera

| Key | Action |
| --- | --- |
| Click | Lock the mouse and enter Fly mode |
| Esc | Release the mouse / close panels |
| W A S D | Move forward and strafe (fly mode) |
| Space | Ascend |
| Shift | Descend |
| Scroll | Change speed tier |
| Tab | Toggle Orbit / Fly mode |

## Markers and tools

| Key | Action |
| --- | --- |
| G | Open the Drop Marker form (fly mode) |
| Right-click | Open the context menu (Drop marker, Measure, Describe) |

## Panels

| Key | Action |
| --- | --- |
| O | Toggle the overview map |
| / | Open the AI Query panel |
| , (comma) | Open Settings |
| Esc | Close the Query panel and clear highlights |

## Help

You can always click the **Help** button in the upper-left to open this window, then use the search box or the sidebar to find what you need.
